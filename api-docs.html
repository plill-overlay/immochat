Immobilienberater AI Static API

This version is designed specifically for GitHub Pages and other static hosts.

How it works

There is only one API file:

https://twalk.chat/api.json

Any LLM or script should fetch that file and filter the listings array locally.

No dynamic endpoints. No query parameters. No backend.

Examples

  • User asks: "available houses in Berlin"
  • LLM fetches: https://twalk.chat/api.json
  • LLM filters locally by city=Berlin, status=available, property_type=house or row-house
